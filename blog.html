1. map, forEach, filter and find there are some differences between them:

map() forEach() filter() find()
Use Array.map() to put the values ​​inside it into a new Array. Array.forEach() work same to map() but he cannot return a value. Array.Filter() can be used to run conditions on the Array and will output all if the conditions match. Array.find() will run the condition inside an array and if the condition is fulfilled, it will take the first value found and not the rest
